# ICC_Cricket_World_Cup_2023_Analysis
## Overview
This project analyzes the **ICC Cricket World Cup 2023** using Python and data analysis libraries. The notebook explores batting, bowling, team performance, and match statistics to generate insights about the tournament.

## Dataset
The analysis uses the following datasets:

- `batting_summary.csv` – Player batting statistics  
- `bowling_summary.csv` – Player bowling statistics  
- `match_schedule_results.csv` – Match results and schedule  
- `world_cup_players_info.csv` – Player information  

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. **Data Loading** – Import datasets using pandas  
2. **Data Cleaning** – Standardize column names and clean text values  
3. **Data Preprocessing** – Convert data types and prepare data for analysis  
4. **Data Integration** – Merge datasets for deeper insights  
5. **Exploratory Data Analysis (EDA)** – Analyze team, batting, and bowling performance  
6. **Visualization** – Generate plots to understand tournament trends

## Key Analysis
- Team performance and total wins
- Batting strength and depth
- Teams scoring **300+ runs**
- Players with most **centuries and half-centuries**
- Bowling performance by teams
- Teams bowling out opponents most frequently
- Venue and match statistics

## Requirements
Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

## How to Run
1. Clone the repository
2. Place all CSV datasets in the same directory as the notebook
3. Run the notebook:

```bash
jupyter notebook ICC_Cricket_World_Cup_2023_Analysis.ipynb
```

## Output
The project generates **visual insights and statistics** about team performance, batting trends, and bowling efficiency during the ICC Cricket World Cup 2023.
